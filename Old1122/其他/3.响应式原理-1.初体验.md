## 2. 响应式

简单说说什么叫做响应式：

- 数据改变视图改变；

- 视图改变数据改变；

```html
<!DOCTYPE html>
<html lang="zh">
  <head>
    <meta charset="UTF-8" />
    <title>lazy_tomato</title>
  </head>

  <body>
    <div id="app"></div>

    <script type="module">
      import Vue from './dist/vue.esm.browser.js'
      new Vue({
        el: '#app',
        template: `
      <div>
        <input type="text" v-model="a">
        <div>a的值： {{a}}</div>
        <button @click="say">点击我改变 a的值</button>
      </div>
      `,
        data() {
          return {
            a: 1,
          }
        },
        methods: {
          say() {
            this.a = '我爱番茄'
          },
        },
      })
    </script>
  </body>
</html>
```
